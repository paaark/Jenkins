Jenkins 란?
===

+ Jenkins는 빌드를 자동화 시키기 위해 사용하는 툴

Jenkins 기능
---

+ 빌드 자동화
+ 테스트 자동화
+ 코드 표준 준수 여부 검사
+ Build Pipeline 구성
+ 다양한 인증 기반과 결합한 인증, 권한 관리 기능
+ 자동화된 배포 관리
